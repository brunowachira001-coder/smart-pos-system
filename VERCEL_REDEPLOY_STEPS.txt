TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

If deployment still fails:

1. Click "View Logs" to see the error
2. Look for the error message
3. Common issues:
   • npm install failed → Check package.json versions
   • Build failed → Check TypeScript errors
   • Deploy failed → Check environment variables

If you see "npm error notarget No matching version found for jsonwebtoken":
   → This means Vercel is still using the old commit
   → Wait 5 minutes and try redeploy again
   → Clear browser cache (Ctrl+Shift+Delete)

If deployment succeeds but pages show 404:
   → Check that outputDirectory is .next in vercel.json
   → Verify buildCommand is npm run build
   → Check that pages exist in smart-pos-system/pages/
