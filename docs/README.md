All SQL schema files are in the `lib/` folder:
- `lib/database.sql` - Main database schema
- `lib/pos-schema.sql` - POS tables
- `lib/customers-step-by-step.sql` - Customers table
- `lib/products-migration.sql` - Products table updates

- `lib/debt-schema.sql` - Debt management tables
- `lib/returns-schema.sql` - Returns tables
- `lib/expenses-schema.sql` - Expenses tables
