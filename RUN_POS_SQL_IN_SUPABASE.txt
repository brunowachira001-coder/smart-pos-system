STEP 2: Copy and Run the SQL
────────────────────────────────────────────────────────────────
1. Open the file: lib/pos-schema.sql
2. Copy ALL the SQL code
3. Paste it into the Supabase SQL Editor
4. Click "Run" or press Ctrl+Enter

STEP 3: Verify Tables Were Created
────────────────────────────────────────────────────────────────
Run this query to check:

SELECT table_name 
FROM information_schema.tables 
WHERE table_schema = 'public' 
AND table_name IN ('cart_items', 'sales_transactions', 'sales_transaction_items');

You should see 3 tables listed.

STEP 4: Test the POS System
────────────────────────────────────────────────────────────────
1. Start your dev server: npm run dev
2. Visit: http://localhost:3000/pos
3. Search for products
4. Add items to cart
5. Complete a test checkout
