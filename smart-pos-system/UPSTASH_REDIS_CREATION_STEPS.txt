📋 WHAT IS UPSTASH REDIS?

  Upstash is a serverless Redis database service. It's like a cache that stores
  data in memory for super-fast access. Your Smart POS System uses it for:
  
  ✓ Session management (keeping users logged in)
  ✓ Rate limiting (preventing abuse)
  ✓ Caching (making searches faster)
  ✓ Real-time data (inventory updates)

     ✓ Click the "Redis" radio button

  2. ENTER DATABASE NAME:
     ✓ Click in the "Name" field
     ✓ Clear any existing text

  3. SELECT REGION:
     ✓ Click the Region dropdown
     ✓ Choose the region closest to you:

     (Choose the same region as your Supabase database for best performance)

  4. SELECT TYPE:
     ✓ Click the Type dropdown
     ✓ Select "Free"
     
     (Free tier is perfect for testing and small deployments)

🆘 TROUBLESHOOTING

PROBLEM: Can't find the Redis URL
SOLUTION:
  1. Click on your database name (smart-pos-redis)
  2. Look for "Connection" section
  3. Scroll down if needed
  4. You should see "Redis URL" with a [Copy] button
  5. If still not visible, refresh the page

PROBLEM: Database not creating
SOLUTION:
  1. Wait 1-2 minutes
  2. Refresh the page (F5 or Cmd+R)
  3. Try creating again
  4. Check if you have free tier available

PROBLEM: Redis URL looks wrong
SOLUTION:
  1. Should start with: redis://default:
  2. Should contain: @us1-example.upstash.io:PORT
  3. Should NOT contain: [brackets] or [placeholders]
  4. If wrong, refresh and copy again

PROBLEM: Can't sign up
SOLUTION:
  1. Try a different sign-up method (GitHub or Google)
  2. Check your email for verification link
  3. Clear browser cache and try again
  4. Use a different browser

📊 YOUR REDIS URL COMPONENTS

  Once you have your Redis URL, here's what each part means:

  redis://default:PASSWORD@endpoint.upstash.io:PORT

  ├─ redis://        = Protocol (Redis)
  ├─ default         = Username
  ├─ PASSWORD        = Your password (keep secret!)
  ├─ endpoint        = Server location
  ├─ upstash.io      = Upstash domain
  └─ PORT            = Connection port

  Example:
  redis://default:AXZ1234567890abcdefghijklmnop@us1-example.upstash.io:12345

  ├─ redis://        = Protocol
  ├─ default         = Username
  ├─ AXZ1234567890... = Your password
  ├─ us1-example     = Your server
  ├─ upstash.io      = Upstash domain
  └─ 12345           = Port number
