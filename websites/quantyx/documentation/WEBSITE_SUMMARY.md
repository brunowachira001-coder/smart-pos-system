## Brand Color Palette

### Primary Colors
- **Electric Blue (#007BFF):** Trust, technology, primary actions
- **Neon Purple (#8A2BE2):** Creativity, energy, secondary accents
- **Dark Gray (#222222):** Professional text, headings, navigation
- **Light Gray (#F8F8F8):** Modern backgrounds, clean aesthetic

### Supporting Colors
- **White (#FFFFFF):** Text on colored backgrounds, highlights
- **Light Blue (#E8F0FF):** Subtle backgrounds, hover states
- **Light Purple (#F3E8FF):** Decorative elements, accents
- **Medium Gray (#666666):** Secondary text, disabled states
- **Light Gray Borders (#E0E0E0):** Dividers, form inputs
- **Dark Blue (#0056B3):** Hover states, visited links
- **Dark Purple (#6A1B9A):** Active states, deep accents

### Brand Gradients
- **Primary Gradient:** `linear-gradient(135deg, #007BFF, #8A2BE2)`
- **Reverse Gradient:** `linear-gradient(135deg, #8A2BE2, #007BFF)`
- **Subtle Gradient:** `linear-gradient(135deg, #F8F8F8, #E8F0FF)`

### Accessibility
- All colors meet WCAG AA/AAA contrast standards
- Colorblind-friendly palette
- Suitable for all users including those with visual impairments
