# QuantyX Digital Technologies - Brand Color Palette

## Executive Summary

A carefully curated color palette designed to communicate trust, innovation, and professionalism. The palette combines electric blue for reliability, neon purple for creativity, with neutral grays for balance and readability.

## Accessibility Guidelines

### Contrast Ratios

| Color Combination | Ratio | WCAG Level | Usage |
|-------------------|-------|-----------|-------|
| #007BFF on #FFFFFF | 8.59:1 | AAA | Primary text, buttons |
| #8A2BE2 on #FFFFFF | 5.51:1 | AA | Secondary text, accents |
| #222222 on #FFFFFF | 18.54:1 | AAA | Body text, headings |
| #222222 on #F8F8F8 | 18.54:1 | AAA | Body text on light bg |
| #007BFF on #F8F8F8 | 8.59:1 | AAA | Links on light bg |
| #FFFFFF on #007BFF | 8.59:1 | AAA | Text on blue bg |
| #FFFFFF on #8A2BE2 | 5.51:1 | AA | Text on purple bg |

### Colorblind Considerations

- **Deuteranopia (Red-Green):** Electric Blue (#007BFF) is highly visible
- **Protanopia (Red-Green):** Use Electric Blue as primary, avoid purple-only indicators
- **Tritanopia (Blue-Yellow):** Avoid blue-yellow combinations
- **Monochromacy (Complete):** Ensure sufficient contrast ratios

**Recommendation:** Always pair colors with patterns, icons, or text labels for critical information.

## Color Implementation

### CSS Variables
```css
:root {
    /* Primary Colors */
    --primary: #007BFF;
    --accent: #8A2BE2;
    --neutral: #222222;
    --background: #F8F8F8;
    
    /* Supporting Colors */
    --white: #FFFFFF;
    --light-blue: #E8F0FF;
    --light-purple: #F3E8FF;
    --medium-gray: #666666;
    --light-gray: #E0E0E0;
    --dark-blue: #0056B3;
    --dark-purple: #6A1B9A;
    
    /* Gradients */
    --gradient-primary: linear-gradient(135deg, #007BFF, #8A2BE2);
    --gradient-reverse: linear-gradient(135deg, #8A2BE2, #007BFF);
    --gradient-subtle: linear-gradient(135deg, #F8F8F8, #E8F0FF);
}
```

### Usage Examples

**Primary Button:**
```css
.btn-primary {
    background-color: var(--primary);
    color: var(--white);
}

.btn-primary:hover {
    background-color: var(--dark-blue);
}
```

**Accent Button:**
```css
.btn-accent {
    background-color: var(--accent);
    color: var(--white);
}

.btn-accent:hover {
    background-color: var(--dark-purple);
}
```

**Text Content:**
```css
body {
    color: var(--neutral);
    background-color: var(--white);
}

a {
    color: var(--primary);
}

a:visited {
    color: var(--dark-blue);
}
```

## Brand Color Guidelines

### Do's ✓
- Use Electric Blue as the primary brand color
- Combine Electric Blue with Neon Purple for visual interest
- Maintain sufficient contrast for accessibility
- Use gradients for modern, dynamic elements
- Apply colors consistently across all mediums
- Use Dark Gray for all body text
- Use Light Gray for backgrounds and subtle elements

### Don'ts ✗
- Don't use colors in reverse (purple as primary)
- Don't reduce color saturation significantly
- Don't use colors without sufficient contrast
- Don't mix with conflicting brand colors
- Don't apply colors to text without proper contrast
- Don't use colors for critical information alone (add patterns/icons)
- Don't change color meanings across mediums
