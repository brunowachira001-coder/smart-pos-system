# Social Media Video Templates

A web-based tool to create stunning video templates for Facebook, Instagram, TikTok, and other social media platforms.

## 🎬 Features

### 6 Ready-to-Use Templates

1. **Product Showcase** 🛍️
   - Highlight your products
   - Add price and description
   - Include call-to-action button
   - Perfect for e-commerce

2. **Customer Testimonial** ⭐
   - Share customer reviews
   - Display ratings
   - Build social proof
   - Increase trust

3. **Flash Sale Countdown** ⏰
   - Create urgency
   - Show discount percentage
   - Countdown timer
   - Drive conversions

4. **Before & After** ✨
   - Show transformations
   - Display results
   - Demonstrate value
   - Increase engagement

5. **Tips & Tricks** 💡
   - Share helpful content
   - Educate your audience
   - Build authority
   - Increase followers

6. **Announcement** 📢
   - Announce new products
   - Promote events
   - Share news
   - Build excitement

## 🚀 How to Use

### Step 1: Open the App
Open `index.html` in your web browser

### Step 2: Choose a Template
Click on any template card to open the editor

### Step 3: Customize
- Fill in your content
- Choose colors
- Preview in real-time
- See changes instantly

### Step 4: Generate
Click "Generate Video" to create your template

### Step 5: Download
The template will download as an image

### Step 6: Create Video
Use the downloaded template in:
- **CapCut** (free, easy)
- **DaVinci Resolve** (free, professional)
- **Canva** (free, simple)
- **Adobe Premiere** (paid, professional)
