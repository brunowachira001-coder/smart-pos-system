🚀 QUICK START (5 MINUTES):

1. npm install
2. cp .env.example .env.local
3. Fill in .env.local with:
   - MongoDB URI (from MongoDB Atlas)
   - Stripe keys (from Stripe Dashboard)
4. npm run dev
5. Open http://localhost:3000
