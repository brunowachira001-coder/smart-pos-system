## Troubleshooting

### "I don't have a GitHub account"
→ Create one at https://github.com (free)

### "I don't have a Stripe account"
→ Create one at https://stripe.com (free, test mode available)

### "I don't have a MongoDB account"
→ Create one at https://www.mongodb.com/cloud/atlas (free tier available)

### "Build failed on Vercel"
→ Check environment variables are correct
→ See ENVIRONMENT_VARIABLES.md for help

### "Site not loading"
→ Wait 5 minutes for deployment to complete
→ Check Vercel dashboard for build status
