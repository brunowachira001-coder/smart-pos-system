## 🚀 Quick Start

### 1. Setup Accounts (15 min)
```
MongoDB Atlas → Get connection string
Stripe → Get API keys
GitHub → Create repository
Vercel → Create account
```

### 2. Configure Variables (5 min)
```
Add to Vercel:
- MONGODB_URI
- JWT_SECRET
- STRIPE_SECRET_KEY
- NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY
- NEXTAUTH_SECRET
- NEXTAUTH_URL
- NEXT_PUBLIC_APP_URL
- NEXT_PUBLIC_ADMIN_EMAIL
- NEXT_PUBLIC_ADMIN_PASSWORD
- NODE_ENV=production
```

### 3. Deploy (10 min)
```
Push to GitHub → Import to Vercel → Deploy
```
