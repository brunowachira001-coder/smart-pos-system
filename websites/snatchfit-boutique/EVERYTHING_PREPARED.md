## Troubleshooting

### "I'm stuck on account setup"
→ Read SETUP_ACCOUNTS.md step by step
→ Follow each step exactly as written

### "I don't know what environment variables to use"
→ Read ENVIRONMENT_VARIABLES.md
→ It explains each variable and where to get it

### "Build failed on Vercel"
→ Check environment variables are correct
→ Verify MongoDB connection string
→ Check Stripe keys are valid

### "Site not loading"
→ Wait 5 minutes for deployment to complete
→ Check Vercel dashboard for build status
→ Clear browser cache

### "Payments not working"
→ Verify Stripe keys are correct
→ Make sure you're using test keys
→ Check Stripe dashboard for errors
