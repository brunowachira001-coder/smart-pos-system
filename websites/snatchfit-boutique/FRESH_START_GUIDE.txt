QUICK OVERVIEW
================================================================================

Step 1: Create Supabase Account (5 min)
  - Go to https://supabase.com
  - Sign up with GitHub or email
  - Verify email

Step 2: Create New Project (5 min)
  - Project name: snatchfit-boutique
  - Generate database password
  - Choose region
  - Wait for creation

Step 3: Create Database Tables (5 min)
  - Go to SQL Editor
  - Copy and paste SQL
  - Run query
  - Tables created!

Step 4: Get Credentials (5 min)
  - Go to Settings → API
  - Copy Project URL
  - Copy Anon Key
  - Copy Service Role Key

Step 5: Verify Tables (2 min)
  - Go to Table Editor
  - Check all 5 tables exist

Step 6: Get Stripe Keys (5 min)
  - Go to https://dashboard.stripe.com
  - Get test API keys
  - Copy Publishable Key
  - Copy Secret Key

Step 7: Generate Secrets (2 min)
  - Run: node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
  - Generate JWT_SECRET
  - Generate NEXTAUTH_SECRET

Step 8: Prepare Variables (5 min)
  - Collect all credentials
  - Save in text file
  - Keep safe!

Step 9: Create GitHub Repo (5 min)
  - Go to https://github.com/new
  - Create repository
  - Push code to GitHub

Step 10: Deploy to Vercel (10 min)
  - Go to https://vercel.com/dashboard
  - Import GitHub repository
  - Add environment variables
  - Deploy!

Step 11: Test Your Site (10 min)
  - Visit Vercel URL
  - Test all features
  - Test checkout

Step 12: Add Products (10 min)
  - Login to admin
  - Add sample products
  - Ready to launch!
