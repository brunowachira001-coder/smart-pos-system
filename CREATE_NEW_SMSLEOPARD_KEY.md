# Create New SMS Leopard API Key

## Steps to Create a New Key

1. **Login to SMS Leopard Dashboard**
   - Go to: https://app.smsleopard.com/
   - Login with your credentials

2. **Navigate to API Keys Section**
   - Look for "API Keys" or "Developer" in the menu
   - Usually under Settings or Account section

3. **Create New API Key**
   - Click "Create New Key" or "Generate API Key"
   - Give it a name like: `Nyla_Wigs_POS_System`
   - Copy the FULL key immediately (it might only show once!)

4. **What to Copy**
   When you create the key, you should see:
   - **API Key** or **Access Token** - This is what we need!
   - It might look like:
     - `slk_live_xxxxxxxxxxxxx` (starts with slk_)
     - OR a long base64 string
     - OR a combination of key:secret

5. **Important: Copy Everything**
   - Copy the ENTIRE key/token
   - Don't copy just part of it
   - Save it somewhere safe (you might not see it again)

## What to Look For in the Documentation

While you're there, also check:

1. **API Documentation Link**
   - Look for "API Docs" or "Developer Guide"
   - Check for example code (curl, PHP, Node.js)

2. **Authentication Format**
   - How do they show authentication in examples?
   - Bearer token? API key header? Basic auth?

3. **Example Request**
   - Copy any example they show for sending SMS
   - This will help us get the exact format
