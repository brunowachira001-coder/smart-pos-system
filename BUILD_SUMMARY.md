## 🚀 Quick Start

### Backend (2 minutes)
```bash
cd backend
npm install
cp .env.example .env
# Update .env with credentials
npm run prisma:generate
npm run prisma:migrate
npm run prisma:seed
npm run dev
```

### Frontend (2 minutes)
```bash
npm install
echo "NEXT_PUBLIC_API_URL=http://localhost:5000/api" > .env.local
npm run dev
```

### Access
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
