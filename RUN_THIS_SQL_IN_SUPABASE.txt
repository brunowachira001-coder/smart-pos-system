To activate the Debt Management system, you need to run the SQL script in your Supabase database.

QUICK STEPS:
1. Go to: https://supabase.com/dashboard
2. Select your project: smart-pos-system
3. Click "SQL Editor" in left sidebar
4. Click "New Query"
5. Open the file: lib/debt-schema.sql
6. Copy ALL the content
7. Paste into Supabase SQL Editor
8. Click "Run" button

WHAT THIS CREATES:
✅ 3 new tables (debts, debt_payments, customer_credit)
✅ Sample debt data (2 customers with debts)
✅ Credit limits for all customers
✅ Automatic triggers for status updates
✅ Indexes for performance

AFTER RUNNING SQL:
Visit: https://smart-pos-system-peach.vercel.app/debt

You should see:
- Outstanding Debt: KSH 1500.00
- 2 active debts
- Total Credit Limit: KSH 115650.00
- Payment recording functionality
