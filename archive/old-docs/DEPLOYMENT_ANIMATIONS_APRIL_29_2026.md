## 📱 Features

### Animation Principles
- **Staggered Delays** - Elements appear in sequence (0.1s - 0.9s)
- **Smooth Easing** - cubic-bezier(0.4, 0, 0.2, 1)
- **Consistent Timing** - 300ms for interactions, 600ms for entrances
- **Subtle Movement** - Small transforms (4-10px)
- **GPU-Accelerated** - Uses transform and opacity for performance

### Hover Effects
- **Buttons:** Scale (1.05x), translate up, shadow glow, icon animations
- **Cards:** Scale (1.05x), translate up (-4px), border intensify, shadow glow
- **Social Icons:** Scale (1.25x), translate up, color change

### Gradients
- **Background:** `from-black via-gray-900 to-black`
- **Title:** `from-indigo-400 via-purple-400 to-pink-400`
- **Buttons:** `from-indigo-600 to-indigo-700`
- **Cards:** Individual color schemes (indigo, purple, pink)
