# SMS Leopard API Key Issue - Let's Fix It

## Current Problem

Account is active but getting "Unauthorized" error.

## Possible Causes

1. **Wrong API key format** - The key might be an access token, not API key
2. **API key not from correct place** - Need to get it from specific location
3. **Different authentication method** - Might use different header format

---

## Where to Get the Correct API Key

### Option 1: API Settings Page

1. Log in to: https://www.smsleopard.com/dashboard
2. Go to **Settings** → **API** or **API Keys**
3. Look for section called:
   - "API Key"
   - "Access Token"
   - "Authentication Token"
   - "Secret Key"

### Option 2: Developer Section

1. In dashboard, look for **Developers** or **API Documentation**
2. Find your API credentials
3. Copy the exact key shown

### Option 3: Account Settings

1. Go to **Account** or **Profile**
2. Look for **API Access** or **Integration**
3. Copy the key from there

---

## What to Look For

The API key might be labeled as:
- ✅ API Key
- ✅ Access Token
- ✅ Secret Key
- ✅ Authentication Token
- ✅ Bearer Token

It might look like:
- `slk_xxxxxxxxxxxxxxxx` (starts with slk_)
- `Bz3LDHWNx1tL6A2BL4ie` (what you have)
- A longer string with more characters
- A JWT token (very long string with dots)
