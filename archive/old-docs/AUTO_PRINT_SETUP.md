# Auto-Print Receipt Setup Guide

## Overview
The system now supports automatic receipt printing. When enabled, the print dialog will automatically open after each transaction completes.

## How to Enable Auto-Print

1. Go to **Settings** page
2. Scroll to the **Preferences** section
3. Toggle **"Auto-Print Receipts"** to ON
4. The setting is saved automatically in your browser

## How It Works

### When Auto-Print is ENABLED:
- After completing a checkout, the receipt preview appears
- After 0.5 seconds, the browser print dialog opens automatically
- Select your printer and click Print
- Or click Cancel to skip printing

### When Auto-Print is DISABLED:
- After completing a checkout, the receipt preview appears
- You must manually click the "Print" button to print
- This gives you time to review the receipt first

## Printer Setup

### For Thermal Printers (80mm):
1. Install the printer driver on your computer
2. Connect the printer via USB or network
3. Set it as your default printer (optional)
4. When the print dialog opens, select your thermal printer
5. The receipt is already formatted for 80mm width

### For Regular Printers (A4):
1. The receipt will print on standard paper
2. You can adjust print settings in the dialog
3. Consider using "Fit to page" or "Scale" options

## Tips

- **First-time setup**: Test with auto-print OFF first to ensure your printer is working
- **Fast checkout**: Enable auto-print for quick transactions
- **Review receipts**: Disable auto-print if you need to review before printing
- **Multiple printers**: The browser will remember your last selected printer

## Troubleshooting

**Print dialog doesn't open automatically:**
- Check that auto-print is enabled in Settings
- Refresh the page and try again
- Some browsers may block automatic print dialogs

**Wrong printer selected:**
- In the print dialog, manually select the correct printer
- The browser will remember your choice for next time

**Receipt doesn't fit on thermal printer:**
- The receipt is pre-formatted for 80mm thermal printers
- If it doesn't fit, check your printer settings
- Ensure "Scale" is set to 100% or "Actual size"

## Browser Compatibility

Auto-print works on:
- Chrome/Edge (recommended)
- Firefox
- Safari

Note: Some browsers may require you to allow popups for the site.
