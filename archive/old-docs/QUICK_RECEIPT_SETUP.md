# Quick Receipt Printing Setup

## ✅ Receipt Printing is Now Active!

After completing a transaction in POS, a receipt preview will automatically appear with a "Print" button.

## 🖨️ Quick Setup (5 Minutes)

### Option 1: Thermal Printer (Recommended)

**What You Need:**
- 80mm thermal receipt printer (e.g., Epson TM-T20, Star TSP143)
- USB cable or WiFi connection

**Setup:**
1. Connect printer to computer (USB or WiFi)
2. Install printer drivers from manufacturer website
3. In POS, complete a sale
4. Click "Print" button
5. Select your thermal printer
6. Done! Receipt prints automatically

**Recommended Printers:**
- **Budget**: Munbyn ITPP047 ($60) - USB/Bluetooth
- **Professional**: Epson TM-T20III ($150) - USB/Ethernet
- **Premium**: Epson TM-T88VI ($250) - USB/Ethernet/Bluetooth

### Option 2: Regular Printer (Any Office Printer)

**Setup:**
1. Make sure printer is connected
2. In POS, complete a sale
3. Click "Print" button
4. Select your printer
5. Receipt prints on A4/Letter paper

### Option 3: Save as PDF (No Printer Needed)

**Setup:**
1. In POS, complete a sale
2. Click "Print" button
3. Select "Save as PDF" or "Microsoft Print to PDF"
4. Save to computer
5. Email to customer or keep for records

## 📱 Mobile/Tablet Setup

**iPad/iPhone:**
- Use AirPrint-compatible printer
- Click Print → Select printer → Print

**Android:**
- Connect to WiFi/Bluetooth printer
- Click Print → Select printer → Print

## 🔧 Common Issues

**Receipt not printing?**
- Check printer is ON and has paper
- Check USB/WiFi connection
- Restart printer
- Try different browser (Chrome recommended)

**Receipt too small/large?**
- In print dialog, set Scale to 100%
- For thermal: Set paper width to 80mm
- For regular: Use A4 or Letter size

**Printer not detected?**
- Install printer drivers
- Windows: Settings → Devices → Printers
- Mac: System Preferences → Printers
- Restart computer
