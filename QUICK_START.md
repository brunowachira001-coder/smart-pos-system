# Smart POS System - Quick Start Guide

## 🚀 Get Started in 5 Minutes

### Step 1: Access the System

Your system is already deployed at:
```
https://smart-pos-system.vercel.app/login
```

### Step 2: Login

Use these credentials:
- **Username**: `admin`
- **Password**: `admin123`

### Step 3: Start Using

#### 📦 Process a Sale (POS)
1. Click **"POS / Sales"** in the sidebar
2. Search for products or browse the list
3. Click **"Add"** to add items to cart
4. Adjust quantities using +/- buttons
5. Select payment method (Cash, Card, M-Pesa)
6. Enter amount paid
7. Click **"Complete Transaction"**

#### 📊 Check Inventory
1. Click **"Inventory / Products"** in the sidebar
2. View all products with stock levels
3. Filter by "Low Stock" to see items that need reordering
4. See total inventory value

#### 👥 Manage Customers
1. Click **"Customers"** in the sidebar
2. Search for customers by name or phone
3. View purchase history and credit usage
4. See total spent per customer

#### 📈 View Reports
1. Click **"Reports / Analytics"** in the sidebar
2. Select time period (Today, This Week, This Month)
3. See sales summary, top products, and trends
4. Download reports (coming soon)

#### 💼 Dashboard Overview
1. Click **"Dashboard"** in the sidebar
2. See today's sales, transaction count, and alerts
3. View recent transactions
4. Check inventory status

## 🆘 Need Help?

### Common Issues

**Q: Login not working?**
- Ensure you're using correct credentials (admin/admin123)
- Check internet connection
- Clear browser cache

**Q: Products not showing?**
- Refresh the page
- Check database connection
- Verify products are in inventory

**Q: Transactions not saving?**
- Check internet connection
- Verify database is accessible
- Try again in a few moments

**Q: Low stock alerts not showing?**
- Refresh the page
- Check reorder points are set
- Verify inventory levels
