What this adds:

  • product_videos table (video demonstrations)
  • product_recommendations_cache (faster recommendations)
  • Background image/video support for shop

Note: The features work without this migration, but you'll only see one image per product (the existing image_url). After migration, you can add multiple angles, lifestyle shots, etc.
